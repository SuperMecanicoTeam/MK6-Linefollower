# MK6-Linefollower

## Harwdare
#### Electrónica
* [Pololu QTR-8RC Reflectance Sensor Array](https://www.pololu.com/product/961)
* 7.4V 2S LiPo Battery
* PCB_V0.1

  * TB6612fng Motor Driver

* Solarbotics Wheels
* 3D printed chassis
* 12v 1000rpm N20 motors
* Plastic ball
